# Architecture Principles

## Clean Architecture

- Separation of concerns
- Domain independence
- Framework isolation

## Hexagonal Architecture

- Ports & Adapters
- Business logic isolation

## Microservices

- Independent deployment
- Database per service (when applicable)
- Communication via REST or messaging

## Event-Driven Design

- Async communication
- Kafka / RabbitMQ usage

## Design Principles

- SOLID
- DRY
- KISS
- YAGNI
